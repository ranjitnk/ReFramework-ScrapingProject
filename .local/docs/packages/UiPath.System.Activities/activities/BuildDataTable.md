# Build Data Table

`UiPath.Core.Activities.BuildDataTable`

Creates a `DataTable` from the schema and sample rows configured in the activity designer.

**Package:** `UiPath.System.Activities`
**Category:** Programming > DataTable

## Properties

### Configuration

| Name | Display Name | Type | Required | Default | Description |
|------|-------------|------|----------|---------|-------------|
| `TableInfo` | Table Info | `string` | Yes | Sample table XML | Serialized table schema and row data generated by the designer. This property is hidden in the property grid. |

### Output

| Name | Display Name | Kind | Type | Description |
|------|-------------|------|------|-------------|
| `DataTable` | Data Table | `OutArgument` | `DataTable` | The `DataTable` created from the configured schema and data. |

## XAML Example

```xml
<ui:BuildDataTable
    DisplayName="Build Data Table"
    DataTable="[dt]" />
```

`xmlns:ui="clr-namespace:UiPath.Core.Activities;assembly=UiPath.System.Activities"`

## Notes

- In Studio, configure columns and rows through the **Build Data Table** designer instead of editing `TableInfo` manually.
- The default activity instance contains a sample table with two columns: `Column1` as `String` and `Column2` as `Int32`.
